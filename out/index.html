    iso-639-1-jp

    言語コードISO-639-1の日本語変換モジュール

    Fork元: meikidd/iso-639-1

    Installation

    npm i iso-639-1-jp

    Usage

    Node.js

    const ISO6391JP = require('iso-639-1-jp');
console.log(ISO6391JP.getName('en')); // '英語'

    ES Module

    import ISO6391JP from 'iso-639-1-jp';
console.log(ISO6391JP.getName('en')); // '英語'

    Browsers

    HTML

    <script type="text/javascript" src="./node_modules/iso-639-1-jp/build/index.js"></script>

    console.log(ISO6391JP.getName('en')); // '英語'

    Development

    Install

    yarn install

    Build

    // Webpack build
yarn build

    mocha test
